# 项目简介

## 核心功能
- 基于YOLOv8的轴承缺陷检测系统，已训练好模型，配置成功环境可直接使用
- 支持1085张轴承缺陷数据集，包括['凹槽','凹陷','擦伤','划痕']4类缺陷识别
- 提供完整源码、完整标注的数据集、训练好的模型及训练结果、项目运行教程
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供用户界面，实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 可自定义置信度、IOU阈值
- 可选择是否显示标签和原图
- 可选择使用不同检测模型

## 适用场景
- 工业轴承产品质量检测
- 制造业生产线自动化缺陷检测
- 设备维护与故障预测
- 机器视觉研究与应用开发
- 教学与研究实验

## 技术特点
- 软件环境：Pycharm+Anaconda（或VSCode+Anaconda）
- 技术栈：Python3.9、OpenCV-Python、PyQt5、Ultralytics、Torch1.9等
- 提供UI界面源文件、图标和完整的程序文件
- 包含数据集图片和项目运行教程
- 界面友好，操作简便，支持多种检测模式切换
